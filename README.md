This is the supporting anlysis code for PJULES comparison including following:
- site metadata loading and site filtering
- NetCDF discovery for meteorological forcing, observations, P-JULES,
  multi-layer JULES, and 13-PFT JULES runs
- daily aggregation of GPP, latent heat, LAI, and FSmc
- half-hourly extraction for diurnal-cycle diagnostics
- unit conversion from model and FLUXNET units to daily carbon flux units
- daily/monthly/yearly merge tables
- GPP and LE skill metrics
- aridity, seasonal, PFT, LAI, and soil-moisture diagnostics
- publication figures and exploratory figures
